#Цель

Разработать веб-сервис "Сокращатель ссылок".

#Требования к программе

1) При вводе URL возвращать короткий URL (классический пример со своим доменом);

2) По короткому URL возвращать оригинальный;

3) Максимальное количество URL, хранимых в программе, будет задаваться пользователем при перезапуске сайта (обновление);

4) Обеспечить удаление URL большего возраста создания при достижении лимита.

#Стек разработки

Python, HTML, Apache.

Для Windows.

#Запууск репозитория

1) Скачать на локальную машину;

2) Перейти в консоли в папку репозитория командой "cd";

3) Запустить локальный сервер командой "python app.py";

4) В консоли прочитать техническую информацию обо всех следующих шагах.
